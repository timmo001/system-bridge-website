Avaliable events are:

- POWER_HIBERNATE
- POWER_LOCK
- POWER_LOGOUT
- POWER_RESTART
- POWER_SHUTDOWN
- POWER_SLEEP

```json
{
    "id": "abc123",
    "token": "abc123",
    "event": "POWER_SLEEP",
    "data": {}
}
```

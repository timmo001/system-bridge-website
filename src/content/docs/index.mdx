System Bridge is an application for your desktop that allows you to monitor and control your system.

## Features

<CardGrid>
  <LandingCard title="System Information">
    <Icon slot="icon" name="mdi:information-outline" />
    Access your system's information via the data modules using the API/WebSocket.
  </LandingCard>
  <LandingCard title="Open files and URLs">
    <Icon slot="icon" name="mdi:cogs" />
    Automate your system by opening a URL/path via the API/WebSocket.
  </LandingCard>
  <LandingCard title="Send Notifications">
    <Icon slot="icon" name="mdi:bell" />
    Send system notifications via the API/WebSocket.
  </LandingCard>
  <LandingCard title="Cross-Platform">
    <Icon slot="icon" name="mdi:desktop-tower-monitor" />
    Compatible with Windows and Linux.
  </LandingCard>
  <LandingCard title="Integrated with Home Assistant">
    <Icon slot="icon" name="mdi:home-assistant" />
    Interact with your Bridges using [Home Assistant](https://www.home-assistant.io/integrations/system_bridge).
  </LandingCard>
  <LandingCard title="Android Companion App">
    <Icon slot="icon" name="mdi:android" />
    Available on the [Play Store](https://play.google.com/store/apps/details?id=dev.timmo.systembridge). Source is on
    [GitHub](https://github.com/timmo001/system-bridge-android-companion).
  </LandingCard>
</CardGrid>

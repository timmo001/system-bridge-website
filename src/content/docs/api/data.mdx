_Get data from data modules._

## Authentication

This endpoint requires authentication via the `X-API-Token` header or the `token` header.

## Request

<Tabs syncKey="auth-header">
  <TabItem label="X-API-Token header">
    ```http
    GET /api/data/{module}
    X-API-Token: {token}
    ```
  </TabItem>
  <TabItem label="token header">
    ```http
    GET /api/data/{module}
    token: {token}
    ```
  </TabItem>
</Tabs>

`{module}` is one of: `battery`, `cpu`, `disks`, `displays`, `gpus`, `media`, `memory`, `networks`, `processes`, `sensors`, `system`.
